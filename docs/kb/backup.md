# Backup & restore

*(On the higher plans.)* Your whole company is a single file on your PC, so backing up is
simple and fast. Use the **Backup & Restore** screen.

## Create a backup
Click **Create Backup**. The app writes a timestamped copy of your company file to the
backups folder. The screen shows a history of backups (time, size).

## Restore a backup
Click **Restore**, pick a backup file, and confirm. The app safety-copies your current data
first, then restores from the chosen backup.

## Automatic reminder
If you haven't backed up in a while (default **7 days**, set in Settings), Accounts HQ
reminds you when you open the app. It keeps the most recent backups and trims the oldest.

## Best practice
- Back up before big operations (migration, bulk edits, reconciliation finalise).
- Copy a backup to a **pen-drive or cloud folder** now and then — the backup file is all you
  need to move your books to another PC.

## Moving to a new computer
Install Accounts HQ on the new PC, copy your company file (or a backup) across, and open it.
Your data is portable because it's just that one file.
