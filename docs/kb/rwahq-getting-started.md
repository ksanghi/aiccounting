# RWA HQ — Getting started (install, create society, Quick Setup)

## Install
Download RWA HQ from the website and run the installer. It's a Windows desktop app.

## Create your society
On first launch, create your society. A **Quick Setup wizard** then runs automatically.

On the **"Your society"** page set:
- Society name
- State code (2-digit, drives GST) and GSTIN if you have one
- The books year (financial year)
- **Whether your units are Flats or Plots** — this labels units throughout the app (the "Flats" screen becomes "Plots", etc.). Set it before adding units.

You can re-run the wizard any time from the **⚙ Setup** button in the status bar, or the tile menu under **Settings → Quick Setup**. It re-opens after a plan upgrade to set up newly unlocked features.

## Finding your way around
Two navigation styles (choose in Settings): a **sidebar** list on the left, or a **tile menu** opened with the **☰ Menu** button. The status bar always shows Menu, the sidebar toggle, the ⚙ Setup button, and your AI Credits balance.
