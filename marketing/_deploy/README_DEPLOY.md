# Marketing site deploy — apps.ai-consultants.in

Generated by `_deploy_prepare.py`.

- **rollback/** — snapshot of the 8 pages currently live (6/8 fetched). Keep this folder; it's your one-click revert.
- **upload/**   — the 8 new pages ready to upload (8/8 staged).

## To deploy (new teal pages)

1. Log in to Hostinger -> Websites -> apps.ai-consultants.in -> File Manager.
2. Navigate to `public_html/` (or wherever the site root maps to).
3. Upload all 8 files from `_deploy/upload/` over the existing ones. Choose **Overwrite** when prompted.
4. Hard-refresh in browser (Ctrl-Shift-R) to confirm:
   - Accent colour is teal (#0EA5A5), not violet.
   - Pricing page shows AHQ tiers from xlsx (20,000 / 50,000 / 100,000 txn).
   - Checkout has 3-product toggle including tradeHQ.

## To roll back (restore the violet pages)

1. Same File Manager + `public_html/`.
2. Upload all 8 files from `_deploy/rollback/` over the live ones. **Overwrite.**
3. Hard-refresh.

That's it — same operation, different folder.

## Notes

- `_deploy/rollback/` was captured at the moment this script ran. **Do not re-run `_deploy_prepare.py` after you've uploaded the new pages** — that would overwrite the rollback with the new files and you'd lose the revert option.
- The 8 pages above are the full marketing site at apps.ai-consultants.in (root). The RWA HQ React SPA at `apps.ai-consultants.in/rwahq/` is a separate bundle and is NOT touched here.
- Files are byte-identical to what's on the server / what's in `marketing/`. No path rewrites.
