GST and TDS, done — not "supported"

State codes drive CGST+SGST vs. IGST automatically. TDS sections apply themselves above the threshold. Returns ready for filing — GSTR-1, GSTR-3B summaries, TDS deduction reports.

  • All standard GST rates (0%, 5%, 12%, 18%, 28%)
  • TDS sections 194A, 194C, 194I, 194J, 194Q built in
  • HSN/SAC capture per item
  • Audit-ready ledger export
